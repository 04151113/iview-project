# iView-project

本工程适用于基于 Vue.js 2 + vue-router + webpack2 + iView 2 的项目，已经将 iView 配置完成，只需要按步骤初始化即可。

## 安装
```bush
// 安装依赖
npm install
```
## 运行
### 开发环境
```bush
// 注意首次使用需要执行下面的init命令来生成入口html文件,以后不用再执行
npm run init
npm run dev
```
### 生产环境(打包)
```bush
npm run build
```
### 访问
在浏览器地址栏输入[http://127.0.0.1:8080](http://127.0.0.1:8080)
